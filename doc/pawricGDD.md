## 1. 游戏概述

### 1.1 游戏定位

* **类型**：像素风动作Roguelike + 宠物养成 + 地牢探险
* **主要玩法**：操控一个可爱的小动物，在地牢中探索、打怪、升级、收集装备、解锁新层。
* **特色卖点**：

  1. 可爱宠物系统（玩家可操纵不同宠物战斗）
  2. 单机/联机皆可
  3. 随机关卡 + 可接任务的自由探索模式
  4. 无数值付费，仅外观付费
  5. 可成长角色系统（角色绑定固定增益，可带入存档）

### 1.2 平台与目标玩家

* **目标平台**：PC(Linux/MacOS/Windows及其他)：Steam, Android及IOS
* **目标玩家群体**：喜欢Roguelike、像素动作、轻冒险、宠物养成的玩家
* **受众参考**：《元气骑士》《绯红森林》《杀戮尖塔》《饥荒》《7DTD》

## 6. 任务与大厅系统

### 6.1 大厅结构

| 区域  | 功能         | 说明 |
| --- | ---------- | --- |
| 任务板 | 接任务、查看进度   | 有柔情猫娘做看板娘 |
| 宠物室 | 管理、喂养、融合宠物 | 可以做成一个GUI 就是玩家换宠物的地方 |
| 商店  | 购买装备、道具    | 大厅的商店和地牢里遇到的商店是差不多的 但是地牢里的商店能卖更多有特色的东西 而且还要支持地牢里的专属货币交易 |
| 传送门 | 进入地牢       | 传送门旁边的布景是不一样的 |
| 休息区 | 恢复、存档      | 玩家从这里上线 与玩家上线下线的机制有关 |

> 休息区：玩家在任务中下线后 可以把玩家保护起来 当玩家再次上线时 会让其在休息区出生 他可以在准备好之后继续上次的任务

大厅有很多个 玩家可以像7dtd“开放贸易路线”那样去到其他的大厅 但前提是要完成一些任务 每个大厅能通往的地牢场景是不同的...

### 6.2 任务系统

* 任务类型：清理 / 收集 / 护送 / 限时 / 精英(类似于7dtd的群聚任务 一倍的怪物)
* 来源：（任务板的柔情猫娘 / NPC / 随机）
* 奖励：（金币 / 装备 / 宠物碎片 / 角色增益）
* 难度与等级：每个地牢有一个等级 1到6级 1级最简单 6级最困难 每级任务仅会在对应等级的地牢出现
* 自由探索模式：无任务也可进入地牢，奖励减少。
